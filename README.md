# Tax Compliance Radar · 税务合规雷达

> 一款面向出海企业的 AI 驱动多国税务合规助手，帮助业务、财务与法务团队在跨境经营中快速识别风险、理解法规、生成可落地的合规建议。

---

## 🌏 我们解决什么问题

企业出海时，每个目的地国家都有自己的税务体系、申报周期、发票规则、跨境政策，信息分散、更新频繁、语言各异。传统做法依赖人工翻阅法规、请教顾问、逐国排查，成本高、时效慢，还容易漏掉关键条款。

**Tax Compliance Radar 把这件事变简单**：只需描述业务情况、选择目标国家，就能一键获得**结构化的合规审计报告 + 可追溯的法规原文 + 逐字流式的智能问答**。

---

## ✨ 核心功能

### 1. 🔍 多国税务合规审计

- **一次填表，多国并行分析**：勾选目标国家（如泰国、越南、印尼、马来西亚、中国等），一次提交即可得到每个国家独立的合规报告
- **动态业务画像**：不同国家展示不同的业务字段（月流水、发票类型、员工规模等），按需填写，不做无关询问
- **结构化风险清单**：每条风险附带风险等级（高/中/低）、影响说明、法规依据
- **可执行建议**：AI 输出的不是模糊建议，而是具体动作项（如"须在 30 日内完成 VAT 登记"）
- **业务场景覆盖**：跨境电商零售、品牌出海直营、外贸综合服务 —— 每种场景内置差异化的合规视角

### 2. 💬 智能合规问答（RAG）

- **有依据的回答**：每个回答都附带引用的法规来源，一键跳转原文
- **来源段落自动定位**：点击引用即打开法规原文并暗金脉冲高亮相关段落，无需自己搜索
- **逐字流式展示**：Token 级渐入动画，回答产生的同时就能开始阅读
- **Think Mode 深度模式**：可选启用更深入的分析，用时间换质量
- **历史可复现**：所有问答自动存档，历史记录中的来源依然可精准跳回原文

### 3. 📄 合同上传自动填表

- **拖拽即用**：上传合同（PDF / DOCX / TXT / MD），系统自动抽取关键字段填入审计表单
- **智能识别**：自动判断涉及的国家、业务类型、金额、发票需求等
- **低置信度提示**：AI 不确定的字段会主动提示人工核对，不做静默填写
- **不落库、不越权**：抽取过程仅用于填表辅助，不留档、不写入用户画像

### 4. 📊 数据看板

- **近 7 日活跃趋势**：审计与问答量走势一图直观
- **风险分布甜甜圈**：高/中/低风险占比一目了然
- **国家审核热度榜**：哪些国家被查得最多
- **最近问答流**：团队最新的合规关注点

### 5. 📚 法规知识库

- **多国原生法规**：泰国 VAT、越南 e-Invoice、印尼 PMK、中国外贸退税、马来西亚 SST 等真实法规原文
- **元数据卡片**：每份法规展示生效日期、适用范围、发布机构等结构化信息
- **别名解析**：中文标题、法规编号、文件名任一方式都能唤起同一份法规
- **持续更新**：新增法规仅需放入知识库，前端引用无需改动

### 6. 📤 报告导出

- **一键 PDF 导出**：审计报告可直接导出为 PDF，方便存档、评审、交付客户
- **AI 内容显式标注**：报告内 AI 生成部分带有明显标识，符合企业审阅规范

### 7. 🎨 使用体验

- **动态 Slogan 提示系统**：等待期间根据所选业务与国家展示相关合规小贴士，不是无意义的转圈
- **底部悬浮问答 Dock**：审计过程中可随时唤出问答窗口追问细节，展开/收起自适应
- **墨黑 + 暗金主题**：低饱和语义色（砖红/暗金/墨绿三档风险），专业、克制、长时间阅读不刺眼
- **胶囊按钮、圆角卡片**：视觉统一，交互一致
- **免责声明常驻**：所有 AI 输出自动附带免责说明，明确"辅助"而非"替代"专业意见

## 🧭 典型使用场景

### 跨境电商运营

> "我们月流水 500 万泰铢，在泰国和越南通过 TikTok Shop 卖货，需要注意什么？"

上传合同或填表 → 两国并行审计 → 得到 VAT 登记门槛、发票要求、代扣代缴风险清单。

### 财务合规排查

> "越南 78 号通函关于电子发票的新规是什么？对我们影响多大？"

直接在问答框提问 → 引用原文回答 → 点击来源查看完整条款。

### 法务尽调

> 客户签约前上传其业务合同

系统自动识别涉及国家与业务模式 → 一键生成多国合规风险简报。

### 管理层周报

数据看板一屏查看：本周团队关注了哪些国家、发现了多少高风险、哪类业务咨询最多。

## 🎯 产品理念

- **有据可查**：AI 不做无来源的断言，每个结论都指向法规原文
- **人机协作**：低置信度主动提示，不掩盖不确定性
- **克制的智能**：不多问一句、不少给一条，只在需要时展开
- **合规优先**：AI 生成内容显式标识，免责声明常驻，边界清晰
