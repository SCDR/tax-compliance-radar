# 印度尼西亚第23条所得税预扣税指南 —— Article 23 Income Tax

> 来源：印度尼西亚税务总局（Direktorat Jenderal Pajak）官方指南

## 一、适用范围

**适用主体**：向以下对象支付特定收入的纳税人（预扣义务人）：
- 股息（支付给持股低于25%的有限责任公司、合作社、国有企业、地方企业）
- 利息（支付给资金出借人）
- 特许权使用费（支付给收款人）
- 奖金、奖励、红利等（支付给非个人主体）
- 租金及其他财产使用收入（土地和/或建筑物租赁除外）
- 工程服务、管理服务、建筑服务、咨询服务及其他未纳入第21条所得税的服务报酬

**豁免情形**（不适用第23条预扣税）：
- 支付给银行的利息（依据《所得税法》第36/2008号第23条第(4)款a项）
- 支付给符合财政部条例的金融服务实体（贷款/融资提供方）的利息（依据《所得税法》第36/2008号第23条第(4)款h项，详见PMK 251/PMK.03/2008）
- 合作社分配给成员的利润（依据《所得税法》第36/2008号第23条第(4)款f项）
- 有限合伙企业、合伙、协会、商号、合资企业成员获得的利润（依据《所得税法》第36/2008号第4条第(3)款i项及第23条第(4)款e项）
- 分配给个人纳税人的股息（适用第4条第(2)款所得税）
- 境内公司纳税人、合作社、国有企业、地方企业获得的股息，且满足：来自留存收益，且持股比例不低于已缴资本的25%（依据《所得税法》第36/2008号第4条第(3)款f项）
- 定期存款利息、储蓄利息（来自银行）、印尼央行证券（SBI）贴现（适用第4条第(2)款所得税）
- 债券利息（适用第4条第(2)款所得税）
- 合作社支付给个人成员的存款利息（适用第4条第(2)款所得税）

## 二、关键门槛与税率

| 序号 | 收入类型 | 税率 × 税基 | 备注 |
|------|----------|-------------|------|
| 1 | 股息 | 15% × 毛额 | 包括任何名称和形式的股息（含保险公司支付给保单持有人的股息）。若收款人无纳税人识别号（TIN），税率提高100%。 |
| 2 | 利息 | 15% × 毛额 | 若收款人无TIN，税率提高100%。 |
| 3 | 特许权使用费 | 15% × 毛额 | 按毛额计算。 |
| 4 | 服务报酬（工程、管理、建筑、咨询及其他） | 2% × 毛额 | 需确认服务类型是否属于PMK-141/PMK.03/2015规定的第23条所得税征税范围。 |
| 5 | 租金（土地和/或建筑物除外） | 2% × 毛额 | 适用于其他财产租赁。 |

**MAP代码与付款类型代码**：
- 股息：MAP Code 411124，Payment Type Code 101
- 利息：MAP Code 411124，Payment Type Code 102
- 特许权使用费：MAP Code 411124，Payment Type Code 103
- 服务报酬：MAP Code 411124，Payment Type Code 104
- 租金（非土地/建筑）：MAP Code 411124，Payment Type Code 100

## 三、申报时限与流程

**预扣义务**：支付时即需预扣第23条所得税，并开具预扣税凭证（使用e-bupot应用程序）。

**缴税时限**：最迟于次月第10日完成缴款。缴款前需通过系统生成账单代码（Billing Code）。

**申报时限**：最迟于次月第20日完成申报。通过登录pajak.go.id网站或应用程序服务提供商（ASP），使用第23条所得税电子申报表（e-bupot）进行申报。

**所需表格/工具**：
- 预扣税凭证：通过第23条所得税e-bupot应用程序开具
- 缴款：使用税务缴款单（SSP）或电子账单
- 申报：第23条所得税申报表（通过e-bupot提交）

## 四、罚则与违规后果

原文未明确提供罚则。但根据一般印尼税法规定，逾期缴税或申报可能面临罚款和滞纳金，具体金额需参考《一般税收规定法》（KUP）及相关实施条例。原文中仅提及若收款人无纳税人识别号（TIN），预扣税率将提高100%。

## 五、常见问答

**问1：哪些股息免征第23条预扣税？**  
答：以下股息免征：支付给持股至少25%的境内公司纳税人、合作社、国有企业、地方企业的股息（来自留存收益）；合作社分配给成员的利润；有限合伙企业等实体成员获得的利润；分配给个人纳税人的股息（适用第4条第(2)款）。

**问2：支付利息给银行是否需要预扣第23条所得税？**  
答：不需要。支付给银行的利息免征第23条预扣税（依据《所得税法》第36/2008号第23条第(4)款a项）。

**问3：预扣税款的支付和申报截止日期是什么？**  
答：支付截止日期为次月第10日，申报截止日期为次月第20日。

**问4：如果收款人没有纳税人识别号（TIN），税率会变化吗？**  
答：会。对于股息、利息、特许权使用费，若收款人无TIN，预扣税率将提高100%（即从15%变为30%）。

**问5：使用服务时如何确定是否适用第23条预扣税？**  
答：需参照PMK-141/PMK.03/2015号条例，确认服务类型是否属于第23条所得税征税范围。若属于，则按2%税率预扣。
