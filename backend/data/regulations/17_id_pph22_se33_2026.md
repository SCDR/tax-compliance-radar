# 印尼进口环节 PPh 22 预扣税：跨境卖家申报口径

> 来源：印尼税务总局 SE-33/PJ/2026 号通函

## 一、税种简介

PPh 22（Pajak Penghasilan Pasal 22）为印尼进口环节的所得税预扣，
适用于所有进口人（含跨境电商跨境包裹）。

## 二、计税基数

**计税基数 = CIF 价 + 关税 + 已代扣 VAT（PPN）**

## 三、税率

| 主体资质 | 税率 |
| --- | --- |
| 持有 **API**（Angka Pengenal Impor）进口识别号 | **2.5%** |
| 未持 API | **7.5%** |
| 未持 NPWP 的个人 | 10% |

## 四、缴纳流程

1. 报关时由海关代收；
2. 海关系统同步入 DJP（税务总局）系统；
3. 卖家在企业所得税年度汇算清缴时可作为**预付税款**抵扣。

## 五、API 申请建议

- 通过 OSS（在线单一提交）平台申请，材料包含公司注册证、贸易许可证、担保金；
- 审批周期约 15–20 个工作日；
- 一次性投入回报显著：以年进口货值 1000 万 USD 为例，
  持 API 与无 API 相比每年节约约 50 万 USD 的现金占用。

## 六、常见误区

1. **PPh 22 与 VAT 是两笔税**，不能混同；
2. VAT（PPN）由平台代扣不代替 PPh 22；
3. PPh 22 缴纳凭证（Bukti Pemotongan）须妥善保存至少 10 年。
