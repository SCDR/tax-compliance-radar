# 关于外国实体在越南经营或取得收入履行纳税义务的指南 —— Circular No.103/2014/TT-BTC dated August 06, 2014

> 来源：MP Law Firm 网站（原文链接待补充）

## 一、适用范围

**适用主体**（第1条）：
- 在越南设有常设机构或未设常设机构的外国商业组织、在越南居住或非居住的外国商业个人（以下简称“外国承包商”和“外国分包商”），根据与越南实体签订的主合同或与外国分包商签订的分包合同在越南经营或取得收入。
- 以国内出口形式在越南提供商品并取得收入的外国实体（如境外公司向越南公司A购买布料，再要求A交付给越南公司B，B向境外公司付款）。
- 在越南从事全部或部分货物分销或服务提供、仍保留货物所有权或承担分销/广告/营销/质量责任或定价的外国实体。
- 通过越南实体谈判或签订合同的外国实体。
- 根据贸易法行使出口、进口、分销、购买出口货物或向越南商人销售货物权利的外国实体。

**豁免情形**（第2条）：
1. 根据《投资法》、《石油法》和《信贷机构法》在越南经营的外国实体。
2. 在越南无附带服务的情况下提供货物的外国实体，且交货条件为：
   - 境外边境交货（卖方承担至境外边境的所有责任和风险）；
   - 越南边境交货（卖方承担至越南边境的所有责任和风险，即使合同包含保修条款）。
3. 在境外提供并使用的服务所取得的收入（如香港港口装卸服务、境外债券发行路演服务等）。
4. 在境外提供的特定服务（如运输工具维修、广告营销（在线广告除外）、贸易促进、境外经纪、培训（在线培训除外）、国际电信费用分摊等）。
5. 使用保税仓库或内陆清关站（ICD）作为国际运输、货物中转或储存待加工货物的外国实体。

**关键定义**（第3条）：
- “主合同”：外国承包商与越南方之间的合同或协议。
- “分包合同”：外国承包商与分包商之间的合同或协议，分包商包括外国分包商和越南分包商。
- “越南领土”：包括陆地、岛屿、内水、领海及上空、领海以外水域（含海床和底土），越南行使完全主权和管辖权的区域。

## 二、关键门槛与税率

**纳税方法选择**：
- **申报法（Declaration Method）**（第8条）：适用于同时满足以下条件的外国承包商/分包商：
  1. 在越南设有常设机构或为越南居民；
  2. 在越南的经营期限自合同生效日起≥183天；
  3. 采用越南会计制度，已申请税务登记并取得税号。
  - 增值税：按《增值税法》及指导文件执行（第9条）。
  - 企业所得税：按《企业所得税法》及指导文件执行（第10条）。

- **直接法（Direct Method）**（第11-13条）：适用于不满足上述任一条件的外国承包商/分包商。越南方需代扣代缴增值税和企业所得税，按固定税率计算。

**直接法下增值税计算**（第12条）：
- 计税基础：含增值税的收入（若合同为不含税价，需转换为含税价）。
- 公式：`应纳增值税 = 应税收入 × 增值税税率`
- 应税收入 = 外国承包商/分包商收到的全部服务及附带服务收入（含越南方代付的税款和费用）。
- 若合同不含增值税，转换公式：`应税收入 = 不含增值税收入 / (1 - 增值税税率)`
- 外国承包商/分包商不得抵扣为执行主合同/分包合同而采购货物/服务的进项增值税。
- **原文未明确提供具体增值税税率**，需参照其他法规（如一般服务适用5%或10%）。

**直接法下企业所得税计算**（第13条，原文截断，但根据上下文）：
- 计税基础：全部货物/服务收入（含越南方代付费用）。
- 若货物交付至越南境内（除第2条第5款情形外）或货物供应附带在越南提供的服务（如安装、试运行、保修等），则企业所得税应税收入为货物/服务的总价值。
- **原文未明确提供具体企业所得税税率**，需参照其他法规（如外国承包商通常适用0.1%-10%的核定利润率，再乘以20%企业所得税率，但具体未在本文中列出）。

**收入类型**（第7条）：
- 应税收入包括：货物/服务收入、财产所有权/使用权转让收入、版权收入、资产转让/清算收入、贷款利息收入、证券转让收入、违约金、其他法定收入。

## 三、申报时限与流程

**代扣代缴义务**（第4条第2款、第5条第1款）：
- 越南方（包括根据越南法律成立的组织、商业实体、外国公司分支机构等）在向外国承包商/分包商付款前，有义务按直接法代扣代缴增值税和企业所得税。
- 具体申报时限和流程：**原文未明确提供**，需参照《税收管理法》（第78/2006/QH11号及修订法）及指导文件。

**税务登记**：
- 采用申报法的外国承包商/分包商需申请税务登记并取得税号（第8条第3款）。
- 采用直接法的外国承包商/分包商无需自行登记，由越南方代扣代缴。

## 五、常见问答

**问1**：外国公司C向越南公司D销售挖掘机，合同约定在越南边境交货，且D承担从边境到目的地的运输，但C提供一年保修。C是否需在越南纳税？
**答**：不需要。根据第2条第2款及例3，该情形属于越南边境交货，且保修不视为在越南提供的附带服务，因此不适用本通知。

**问2**：香港公司H为越南公司A在香港港口提供装卸服务，A向H支付费用。H是否需在越南纳税？
**答**：不需要。根据第2条第3款及例4，该服务在境外提供并消费，不属于越南应税收入。

**问3**：外国承包商在越南设有常设机构，经营期限超过183天，且采用越南会计制度。应如何纳税？
**答**：该承包商符合申报法条件（第8条），应按《增值税法》和《企业所得税法》正常申报纳税，可抵扣进项税。

**问4**：越南公司B与外国承包商A签订合同，合同金额不含增值税，但B需承担A的增值税。如何计算应税收入？
**答**：根据第12条第b.1款，应税收入 = 不含增值税收入 / (1 - 增值税税率)。例如，不含税收入30万美元，增值税税率5%，则应税收入 = 300,000 / (1-5%) ≈ 315,789.47美元。

**问5**：外国承包商Z向越南公司E提供原材料，E加工后交付给越南公司G，G再加工后返还给Z，Z向G支付加工费。Z是否适用本通知？
**答**：不适用。根据第1条第2款例3，该情形属于境外公司提供原材料、越南公司加工后返还，Z不直接向越南公司销售货物，因此不受本通知约束。
