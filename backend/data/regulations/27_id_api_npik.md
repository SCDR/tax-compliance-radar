# 印尼贸易部进口商识别号（API / NPIK）规定

> 来源：印尼贸易部第 75/2018 号部长条例（Permendag 75/2018）及其后续修订；
> 印尼海关（DJBC）与贸易部联合执行，任何进口至印尼的商业货物均须由持证进口商完成清关。

## API 分类

- **API-U（Angka Pengenal Importir – Umum）**：一般进口许可
  - 面向贸易商，可进口多品类销售型商品；
  - 商品品类不受限，但**同一 API-U 下须归属同一 HS Code 4 位分组**；
- **API-P（Angka Pengenal Importir – Produsen）**：生产商进口许可
  - 面向制造企业，仅可进口自用原材料、辅料、设备；
  - **不得转售**进口货物，违者吊销 API 并列入黑名单。

## NPIK（Nomor Pengenal Importir Khusus）

对特定敏感品类叠加要求**特别进口商识别号**：

- 电子产品（手机、笔记本、家电）；
- 服装、鞋类、纺织品；
- 食品、饮料、玩具；
- 机械与备件。

NPIK 与 API 联合使用，缺一不可。

## 申请条件

- 在印尼注册的**本地法人**（PT / PT PMA）；
- 具备 NPWP 税号与 PKP（应税企业）资质；
- 实缴资本 ≥ **IDR 10 亿**（PT PMA 通常需 100 亿）；
- 具备仓储、销售或生产实地经营场所并提供照片证明。

## 跨境电商的常见路径

境外卖家 **无法直接** 持有 API/NPIK，通常通过以下三种方式合规进口：

- **委任印尼进口商**：由持有 API-U + NPIK 的印尼贸易商作为进口方；
- **设立 PT PMA**：外资在印尼注册子公司，取得 API-U 后自主进口；
- **PMSE 跨境直邮**：单件货值 ≤ FOB USD 3 且不属敏感品类，可通过跨境快件通道，
  由 PJT（快件运营人）代办清关——**不占用 API 名额**，但仍须按 PMSE 规则代征 VAT。

## 违规后果

- **无 API 进口**：整批货物退运或没收，处货值 **1–2 倍罚款**；
- **API 品类错配**：货物扣押，最长 30 天调查；
- **NPIK 缺失**：敏感品类无法清关，滞港产生 IDR 数百万/天的滞箱费；
- **API-P 转售**：吊销 API 并列入黑名单，负责人被禁止 5 年内担任印尼公司董事。
