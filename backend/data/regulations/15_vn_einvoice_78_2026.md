# 越南电子发票新规（78/2026/TT-BTC）

> 来源：越南财政部与税务总局 2026-06-15 联合发布 78/2026/TT-BTC 号通知

## 一、适用范围

自 **2026-09-01** 起，所有面向越南个人消费者销售商品或服务的境内外电商主体（B2C）
均须开具越南电子发票（e-invoice）。覆盖：

- 本地商户与境外注册纳税人；
- Shopee、Lazada、TikTok Shop 越南站；
- DTC 独立站品牌方。

## 二、认证服务商（VAN）

越南税务总局认证的 8 家电子发票服务商：

1. **Viettel** — 电信运营商方案，接入案例最多；
2. **VNPT** — 与 ERP 集成能力强；
3. **FPT** — 面向大型电商与金融；
4. Misa、BKAV、EFY、TS24、CyberLotus。

## 三、发票内容与流程

1. 卖家在售出商品时通过 VAN API 请求生成 e-invoice；
2. 越南税务总局系统自动分配发票号；
3. 发票 PDF/XML 同步给消费者（邮箱/短信/App 内消息）；
4. 卖家保留 XML 原文件 10 年备查。

## 四、罚则

| 情形 | 罚款 |
| --- | --- |
| 单张漏开/晚开 | **400,000 – 800,000 VND**（约 16 – 32 USD） |
| 伪造发票 | 视为逃税，追缴税款 + 3 倍罚款 |
| 年内累计漏开 ≥ 100 张 | 列入税务黑名单，暂停平台经营权限 |

## 五、DTC 品牌方接入建议

1. 优先选择 Viettel 或 VNPT，兼顾稳定性与接入速度；
2. 与自有 ERP / 独立站订单系统联调，实现下单即开票；
3. 部署重试与对账机制，避免网络抖动导致漏开。
