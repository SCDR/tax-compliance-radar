# 越南 78/2026/TT-BTC 号通知

> 关联主文档：[[越南电子发票新规]]

## 一、通知概况

- 编号：**78/2026/TT-BTC**
- 联合发布：越南财政部（Bộ Tài chính）、越南税务总局（Tổng cục Thuế）
- 发布日期：2026-06-15
- 生效日期：**2026-09-01**

## 二、适用范围

- 面向越南个人消费者销售商品或服务的境内外电商主体（B2C）
- 覆盖 Shopee VN、Lazada VN、TikTok Shop VN 及 DTC 独立站
- 与 78/2021/TT-BTC 相比：将强制电子发票适用范围由 B2B 扩展至 B2C

## 三、发票要求

- 每笔 B2C 订单须开具电子发票，格式遵循国家统一 XML schema；
- 通过越南税务总局认证的 8 家 VAN（Viettel、VNPT、FPT 等）提交；
- XML 原文件须保留 **10 年** 备查。

## 四、罚则

| 情形 | 罚款 |
| --- | --- |
| 单张漏开/晚开 | 400,000 – 800,000 VND |
| 伪造发票 | 追缴税款 + 3 倍罚款 |
| 年内累计漏开 ≥ 100 张 | 列入税务黑名单，暂停平台经营权限 |

## 五、DTC 品牌接入建议

- 优先 Viettel 或 VNPT，稳定性与接入速度兼顾；
- 与自有 ERP、独立站订单系统联调，实现下单即开票；
- 部署重试与对账机制，避免网络抖动漏开。
