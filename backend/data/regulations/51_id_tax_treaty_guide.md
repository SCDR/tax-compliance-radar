# 印尼避免双重征税协定（P3B）适用程序指南 —— Procedures for The Application of Double Taxation Avoidance Agreement (P3B)

> 来源：Direktorat Jenderal Pajak（印尼税务总局）官方发布

## 一、适用范围

- **适用主体**：非居民纳税人（WPLN），即根据印尼所得税法规定，从印尼取得或获得收入的境外税务主体，无论是否通过印尼常设机构。
- **豁免情形**：原文未明确提供豁免情形。但享受协定待遇需满足以下条件：
  1. 收入获得者不是印尼居民税务主体；
  2. 收入获得者必须是协定对方国家或管辖区的居民个人或实体；
  3. 不存在滥用协定（P3B）的情形（依据PER-25/PJ/2018第5条）；
  4. 若协定要求，收入获得者必须是受益所有人。

## 二、关键门槛与税率

- **关键门槛**：享受协定待遇需提交有效的SKD WPLN（居民身份证明），且需满足受益所有人条件。
- **受益所有人条件**：
  - 对于个人WPLN：不得作为代理人或名义持有人。
  - 对于企业WPLN：必须同时满足以下5项条件：
    1. 不得作为代理人、名义持有人或导管公司；
    2. 对产生印尼收入的资金、资产或权利拥有控制权或受益权；
    3. 该实体不超过50%的收入用于履行对其他方的负债；
    4. 承担其拥有的资产、资本或负债的风险；
    5. 无论书面或非书面，无义务将部分或全部从印尼取得的收入转给其他方。
- **税率**：原文未明确提供具体协定税率，需根据具体P3B协定条款确定。

## 三、申报时限与流程

- **SKD WPLN提交流程**：
  1. WPLN向扣缴义务人提交符合要求的SKD WPLN（使用DGT表格，填写正确、完整、清晰，由WPLN签字或同等签名，并由主管当局合法化，包含无滥用协定声明及受益所有人声明）。
  2. 扣缴义务人收到SKD WPLN后，须通过税务总局网站或指定渠道将SKD WPLN中的电子信息传送给税务总局。
  3. 扣缴义务人向WPLN出具SKD WPLN接收凭证。
  4. SKD WPLN仅需提交一次，在SKD WPLN有效期内可重复使用。
  5. 已持有SKD WPLN接收凭证的WPLN，在有效期内后续扣缴时无需再次提交SKD WPLN。
  6. WPLN须将SKD WPLN接收凭证提交给扣缴义务人。
- **扣缴义务人核查与申报**：
  1. 扣缴义务人收到接收凭证后，须通过税务总局网站或指定渠道核查SKD WPLN信息。
  2. 若核查发现未满足协定条件，扣缴义务人须按印尼所得税法规定进行扣缴。
  3. 扣缴义务人须制作电子扣缴凭证（或纸质凭证）。
  4. 扣缴义务人须按规定申报扣缴情况（纳税申报表）。
  5. 即使根据协定无需在印尼扣缴税款，扣缴义务人仍须在纳税申报表中报告该笔收入。

## 四、罚则与违规后果

原文未明确提供罚则、滞纳金比例或刑事责任。但若扣缴义务人未按规定核查或申报，可能适用印尼一般税务罚则（原文未详述）。

## 五、常见问答

**Q1: 什么是SKD WPLN？**
A: SKD WPLN是非居民纳税人的居民身份证明，需使用DGT表格，经主管当局合法化，并包含无滥用协定及受益所有人声明。

**Q2: SKD WPLN需要每年提交吗？**
A: 不需要。SKD WPLN仅需提交一次，在有效期内可重复使用，后续扣缴时只需提交接收凭证。

**Q3: 如果扣缴义务人核查发现WPLN不满足协定条件怎么办？**
A: 扣缴义务人须按印尼所得税法规定进行扣缴，不得适用协定优惠税率。

**Q4: 即使根据协定无需扣税，扣缴义务人还需要报告吗？**
A: 需要。扣缴义务人仍须在纳税申报表中报告该笔收入。

**Q5: 企业WPLN满足什么条件才能被视为受益所有人？**
A: 需同时满足：非代理人/名义持有人/导管公司；对产生收入的资产有控制权；不超过50%收入用于偿债；承担资产风险；无转付义务。
