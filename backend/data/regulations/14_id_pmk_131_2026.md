# 印尼 PMK-131/2026：跨境电商增值税征收细则

> 来源：印尼财政部（Kementerian Keuangan）第 PMK-131/2026 号部长条例

## 一、政策要点

1. **取消低值免税**：自 2026-07-01 起，所有进入印尼的跨境电商包裹取消 3 美元以下 VAT 豁免；
2. **税率**：一律按 **11%** 征收 PPN；
3. **代扣代缴**：通过持牌电商平台（Tokopedia、Shopee ID、Lazada ID 等）销售的，
   由平台在下单时代扣代缴；
4. **DTC 独立站**：品牌方须自行注册 **NPWP**（印尼税务身份号）与 **PPN 号**，并按月申报 **SPT Masa PPN**。

## 二、独立站合规义务

| 项目 | 要求 |
| --- | --- |
| NPWP 注册 | 通过 OSS 平台在线申请，约 3 个工作日 |
| PPN 号 | NPWP 下账后自动开通 |
| 月度申报 | 每月 20 日前提交 SPT Masa PPN |
| 电子发票 | 使用 e-Faktur 系统开具税务发票 |
| 银行账户 | 建议在印尼开设本地账户便于税款划扣 |

## 三、罚则

- **未合规包裹**：海关直接扣押，货主自提日起 30 日内未处理的视为放弃；
- **货值罚款**：处货值 **50%–100%** 罚款；
- **补税**：未申报 VAT 视同偷漏税，按追缴金额 2%/月计息。

## 四、对品牌方的建议

1. 尽快通过本地代理机构完成 NPWP + PPN 注册；
2. 独立站前端根据地区自动含税报价，避免将 11% 税负一次性转嫁给消费者；
3. 评估是否申请 **API 进口识别号** 以适用更低的 PPh 22 预扣税率（详见 [[印尼PPh22预扣税]]）。
