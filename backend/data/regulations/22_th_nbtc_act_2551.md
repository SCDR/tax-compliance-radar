# 泰国电信设备法案 B.E. 2551（2008）

> 来源：泰国国家广播和电信委员会（NBTC）依据《电信业务法》与《无线电通信法》授权发布，
> 版本以 NBTC 官网 2026 年最新公告为准。

## 适用范围

以下品类进口至泰国销售、租赁或分发前，须取得 NBTC 颁发的类型认证（Type Approval）
或符合性声明（SDoC）：

- 手机、平板、可穿戴设备等具备蜂窝网络能力的终端；
- 路由器、蓝牙音箱、Wi-Fi 摄像头等无线收发设备；
- 遥控玩具、无人机（含航模）、车载对讲机等业余频段设备。

## 认证义务

- **注册主体**：进口商或制造商须为在泰国注册的法人，或委托 NBTC 授权的本地代理；
- **认证时限**：首批货物到港前完成认证，认证周期通常 4–8 周；
- **样品送检**：需向 NBTC 认可的实验室送 2–3 台样机进行射频与 EMC 测试；
- **标识义务**：认证通过后必须在产品外壳或包装印制 NBTC ID 标签。

## 违规后果

- **无证进口**：整批货物在海关被扣押，进口商列入 NBTC 黑名单；
- **无证销售**：处 **10 万泰铢以下罚款**，或 **6 个月以下监禁**，或两者并罚；
- **虚假标识**：加倍处罚并公告违规主体名单。

## 与跨境电商的关联

- 跨境电商卖家须在 Shopee / Lazada / TikTok Shop 类目审核时上传 NBTC 认证编号；
- TikTok Shop 泰国站自 2025-08 起对未提供 NBTC ID 的电子产品链接强制下架；
- 无本地实体的境外卖家须委任 NBTC 备案的本地代理承担合规责任。
