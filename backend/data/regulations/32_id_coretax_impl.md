# Coretax DJP税务管理系统实施指南 —— Implementation of Coretax DJP

> 来源：印度尼西亚税务总局（Direktorat Jenderal Pajak）官网 https://www.pajak.go.id/reformdjp/coretax

## 一、适用范围

- **适用主体**：所有在印度尼西亚负有纳税义务的纳税人，包括：
  - 已注册DJP Online账户的纳税人（已有电子税务账户）
  - 已拥有纳税人识别号（NPWP）但从未使用过DJP Online的纳税人
  - 尚未拥有NPWP的新纳税人
- **豁免情形**：原文未明确提供。

## 二、关键门槛与税率

原文未涉及税率、税基或计算方式等具体税务门槛。Coretax系统为税务行政管理平台，不直接规定税率或门槛。

## 三、申报时限与流程

### 系统上线时间
- Coretax DJP系统由总统普拉博沃·苏比安托（Presiden Prabowo Subianto）于 **2024年12月31日** 正式启动。
- 自 **2025年1月** 起，所有税务行政管理（注册、申报SPT、缴款、其他服务）均通过Coretax系统办理。

### 三类纳税人访问流程

| 纳税人类型 | 操作步骤 |
|-----------|---------|
| **已有DJP Online账户** | 通过Coretax登录页面的“忘记密码”（Lupa Kata Sandi）菜单重置密码。重置链接将通过电子邮件或短信发送至纳税人登记的邮箱和手机号。 |
| **有NPWP但无DJP Online账户** | 通过Coretax的“纳税人账户激活”（Aktivasi Akun Wajib Pajak）菜单提交激活申请。填写表格并确保提供有效、可用的手机号和邮箱。 |
| **新纳税人（无NPWP）** | 通过Coretax的“在此注册”（Daftar di Sini）菜单进行NPWP注册。按照指引填写注册表格，完成后即可使用Coretax。 |

### 所需表格
- 重置密码：无需表格，通过系统菜单操作。
- 激活账户：在线填写激活申请表。
- 新注册：在线填写NPWP注册表。

## 四、罚则与违规后果

原文未明确提供任何罚款、滞纳金或刑事责任规定。建议纳税人遵守Coretax系统使用要求，确保登记信息（邮箱、手机号）真实有效。

## 五、常见问答

**Q1: 我已经有DJP Online账户，如何登录Coretax？**  
A: 直接访问Coretax系统，点击“忘记密码”菜单，按提示重置密码。重置链接会发送到您登记的邮箱或手机。

**Q2: 我有NPWP但从未用过DJP Online，能使用Coretax吗？**  
A: 可以。通过Coretax的“纳税人账户激活”菜单提交激活申请，填写信息并提交后即可使用。

**Q3: 我是新纳税人，如何注册NPWP？**  
A: 在Coretax系统中选择“在此注册”菜单，按照指引填写注册表格，完成后即可获得NPWP并使用Coretax。

**Q4: 如果收不到重置密码的链接怎么办？**  
A: 请确保您登记的邮箱和手机号有效且可访问。若仍无法收到，可联系Kring Pajak 1500200或通过pajak.go.id在线客服、邮箱 informasi@pajak.go.id 或 pengaduan@pajak.go.id 寻求帮助。

**Q5: Coretax系统是否支持所有税务申报类型？**  
A: 原文指出Coretax涵盖从注册、SPT申报、缴款到其他服务的全部税务行政管理，自2025年1月起全面使用。具体申报类型可参考官方指南。
