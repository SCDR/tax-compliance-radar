# 印尼财政部 PMK-131/2026 号部长条例

> 关联主文档：[[印尼PMK-131跨境电商]]

## 一、条例概况

- 签发机构：Kementerian Keuangan Republik Indonesia（印尼财政部）
- 编号：**PMK-131/2026**
- 生效日期：**2026-07-01**
- 前置文件：PMK-190/2022（本次修订对象）

## 二、修订要点

1. **取消低值免税**：3 美元以下跨境包裹不再享受 VAT（PPN）豁免；
2. **税率**：一律按 **11%** 征收；
3. **代扣代缴**：通过持牌电商平台（Tokopedia、Shopee ID、Lazada ID 等）销售的，由平台在下单时代扣代缴；
4. **境外独立站**：须自行注册 NPWP 与 PPN 号，按月提交 SPT Masa PPN。

## 三、罚则

- 未合规包裹：海关扣押；
- 货主 30 日内未处理的：视为放弃；
- 处罚：**货值 50%–100%** 罚款；
- 未申报 VAT：视同偷漏税，追缴金额按 2%/月计息。
