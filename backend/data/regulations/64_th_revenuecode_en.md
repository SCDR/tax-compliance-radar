# 泰国增值税法第四章 —— Chapter 4 Value Added Tax

> 来源：泰国税务局英文官网

## 一、适用范围

- **应征增值税的业务**（Section 77/2）：明确哪些业务须缴纳增值税。
- **免征增值税的业务**（Section 77/3）：列举不适用增值税的业务类型。
- **业务性质判定**（Section 77/5）：区分销售货物与提供服务。
- **小企业豁免**（Section 81/1）：符合皇家法令规定的小企业可免于增值税登记。
- **自愿登记**（Section 81/3）：即使符合豁免条件，仍可申请增值税登记。
- **合同价值门槛**（Section 77/4）：合同价值不低于500,000泰铢的，须提交合同副本。

## 二、关键门槛与税率

| 条款 | 内容 |
|------|------|
| Section 80 | 标准增值税税率 **10.0%** |
| Section 80/1 | 适用零税率的业务（如出口、国际运输等） |
| Section 80/2 | 特定业务适用税率 **2.5%** |
| Section 77/4 | 合同价值不低于 **500,000 泰铢** 须提交副本 |
| Section 81/1 | 小企业门槛（具体金额由皇家法令规定，原文未明确） |

- **税基计算**（Part 3）：以收到的或应收的总价值为税基（Section 79），进口货物税基按海关价值计算（Section 79/2），外币换算规则（Section 79/4）。
- **进项税抵扣**（Section 82/3）：允许从销项税中抵扣进项税，但存在不可抵扣情形（Section 82/5）。

## 三、申报时限与流程

- **申报周期**：
  - 按月申报（Section 83）：增值税注册人须每月提交申报表并缴税。
  - 部分类别可定期申报，周期不超过3个月（Section 83/1）。
- **增值税登记**（Part 9）：
  - 须在开业前申请登记（Section 85）。
  - 登记时限（Section 85/1）。
  - 变更、暂停、终止、合并等情形须通知税务局（Section 85/6–85/19）。
- **税务发票**（Part 10）：
  - 须开具税务发票（Section 86），发票内容要求（Section 86/4）。
  - 禁止非注册人开具税务发票（Section 86/13）。
  - 借项通知单（Section 86/9）、贷项通知单（Section 86/10）。
- **报告与记录保存**（Part 11）：
  - 须编制销项税报告、进项税报告及货物/原材料报告（Section 87）。
  - 保存税务发票、报告及其他文件（Section 87/3）。

## 四、罚则与违规后果

- **罚款与附加费**（Part 13）：
  - 附加费（Section 89/1）：原文未明确比例，通常为未缴税款的固定比例。
  - 罚款与附加费视为增值税的一部分（Section 89/2）。
- **刑事处罚**（Part 14）：

| 条款 | 处罚内容 |
|------|----------|
| Section 90 | 罚款不超过 **2,000 泰铢** |
| Section 90/1 | 罚款不超过 **5,000 泰铢** |
| Section 90/2 | 监禁不超过 **1 个月** 或罚款不超过 **5,000 泰铢**，或两者并罚 |
| Section 90/3 | 监禁不超过 **6 个月** 或罚款不超过 **10,000 泰铢**，或两者并罚 |
| Section 90/4 | 监禁 **3 个月至 7 年**，并处罚款 **2,000 至 200,000 泰铢** |
| Section 90/5 | 董事、经理或代表人员承担相应刑事责任 |

- **评估与核定**（Part 12）：税务官员有权核定增值税、罚款及附加费（Section 88），可发出传票、进入场所检查（Section 88/3–88/4）。

## 五、常见问答

原文未提供具体问答。以下为基于条款标题的常见问题示例（非原文内容，仅供参考）：

1. **哪些业务需要缴纳增值税？**  
   根据 Section 77/2，所有在泰国境内销售货物或提供服务的业务原则上均须缴纳增值税，但 Section 77/3 列明了豁免情形。

2. **增值税税率是多少？**  
   标准税率为 10%（Section 80），目前实际执行税率可能低于此（原文未明确）。出口等业务适用零税率（Section 80/1），特定业务适用 2.5%（Section 80/2）。

3. **小企业是否需要登记增值税？**  
   符合皇家法令规定的小企业可免于登记（Section 81/1），但可自愿申请登记（Section 81/3）。

4. **增值税申报周期是多久？**  
   一般按月申报（Section 83），部分类别可申请最长 3 个月的定期申报（Section 83/1）。

5. **未按时申报或缴税会有什么后果？**  
   可能面临罚款（Section 90–90/4）、附加费（Section 89/1），严重情形可导致监禁（最高 7 年）及高额罚金（Section 90/4）。
