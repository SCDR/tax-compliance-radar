# TikTok Shop 泰国站 VAT 合规操作指引

> 来源：TikTok Shop 泰国站卖家公告（2026-05-20）；泰国税务局跨境电商合规要求

## 一、生效日期

**2026 年 6 月 15 日** 起，所有 TikTok Shop 泰国站卖家须在店铺后台完成 VAT 信息绑定。

## 二、绑定要求

1. **VAT 注册证书**：上传泰国税务局出具的 P.P.20 证书扫描件；
2. **VAT ID**：填写 13 位泰国税号；
3. **e-Tax Invoice 服务商**：达到 VAT 门槛的卖家须填写已对接的 e-Tax Invoice 服务商与 API endpoint；
4. **联系人**：财税负责人姓名与邮箱。

## 三、未合规处理

| 违规情形 | 处理方式 |
| --- | --- |
| 未绑定 VAT ID | For You 页与 LIVE 场景推荐权重下调 80% |
| 绑定信息造假 | 立即下架商品并冻结提现 3 个工作日 |
| 多次警告未整改 | 暂停开播权限直至补齐资料 |

## 四、平台稽核机制

- 平台每周向泰国税务局同步一次 VAT 绑定状态；
- 税务局如发现异常，可通过平台直接下发整改通知；
- 合规完成的店铺会在卖家后台首页显示"合规勋章"。

## 五、操作路径

Seller Center → Business Center → Tax Settings → 上传 VAT 证书 → 填写 VAT ID → 提交审核（3 个工作日）。
