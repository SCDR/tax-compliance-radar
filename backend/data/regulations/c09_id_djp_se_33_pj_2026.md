# 印尼税务总局 SE-33/PJ/2026 号通函

> 关联主文档：[[印尼PPh22预扣税]]

## 一、通函概况

- 编号：**SE-33/PJ/2026**
- 签发机构：Direktorat Jenderal Pajak（印尼税务总局，DJP）
- 发布日期：2026-05
- 目的：澄清 PPh 22 进口环节预扣税针对跨境电商的适用口径。

## 二、计税基数

**基数 = CIF + 关税 + 已代扣 VAT（PPN）**

## 三、税率适用

| 主体资质 | 税率 |
| --- | --- |
| 持 **API**（Angka Pengenal Impor）进口识别号 | **2.5%** |
| 未持 API | **7.5%** |
| 未持 NPWP 的个人进口 | 10% |

## 四、缴纳流程

- 报关时由海关代收；
- 海关系统同步至 DJP 系统；
- 卖家在企业所得税年度汇算清缴时可作为**预付税款**抵扣。

## 五、常见误区

1. PPh 22 与 VAT（PPN）是两笔税，不能混同；
2. 平台代扣 VAT 不代替 PPh 22；
3. PPh 22 缴纳凭证（Bukti Pemotongan）须至少保存 **10 年**。
